# Analytical_Solution_Linear_Regression

Short Jupyter Notebook demonstrating the analytical solution for linear regression.

Obtaining the least square solution involves: 

- **Arrays** 
- **Transposition**.
- **Matrix inversion**.
- **Matrix multiplication**.
